how to add lis this readme

To add a profile README like the one in your screenshot, you need to create a special repository whose name is exactly your GitHub username.

Step 1: Create a new repository
Go to GitHub.

Click + → New repository.

Repository name must be exactly your GitHub username.

If your username is nayanbobde, the repository name must be:

nayanbobde
Make it Public.

Check ✅ Add a README file.

Click Create repository.

GitHub will automatically display this README on your profile page.

Step 2: Edit the README
Open the repository.

Click README.md.

Click the ✏️ Edit button.

Replace the default content with your Markdown.

Click Commit changes.

Step 3: Add GitHub Stats (Optional)
Replace YOUR_USERNAME with your GitHub username.
